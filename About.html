Introduccion
En el siguiente documento hacemos presentación del proyecto de programación web al que titulamos “tercer tiempo”. Esto con el fin de mostrar el objetivo de la página, las herramientas de desarrollo, la planeación, roles de desarrollo entre otras actividades.


Objetivo general
Desarrollar una página web que integre todo el contenido relacionado con la materia de “Programación Web” de libre diseño y funcional en la mayor parte de sus atributos. Haciendo uso primordial de las herramientas de desarrollo HTML para la estructura de diseño de página web junto con CSS y programación java script para funcionalidad principal


Objetivos específicos

• Implementación de JavaScript del lado del cliente

• Implementación de JavaScript del lado del servidor

• Diseño de un sitio web con HTML 5

• Implementación de CSS en el diseño de HTML

• Desarrollo de una página web funcional

Requisitos Recomendados

-teclado, mouse, y conexión de internet

-Descripción de la Pagina Web : Se trata de un sitio orientado a las noticias de futbol. Cuenta con una base de datos con la intención de almacenar en sus registros avances de equipos, partidos, noticas, etc. Básicamente aspectos relevantes de un noticiero deportivo en línea, desde fechas próximas, tablas de resultados y consulta de contenido sobre equipos. La página web cuenta con una interfaz sencilla que permite una comprensión rápida del mapa de navegación.
